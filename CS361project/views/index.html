Language Translation Game

RULES:

This app is used to practice your skills translating English into Spanish. Just below is text in the English language that you will attempt to translate. Once your anser is submitted your answer will be translated so you can compare it. Then you can see exactly where you're off without giving the answer away for later attempts. Then you can click play again to try multiple times. Enjoy!


This application can be used for many languages after selecting try again it will always default back to Spanish.
